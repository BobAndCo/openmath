binary operations and groups

binary operation
Let `S` be a set, a binary operation on `S` is a function `star : S xx S to S` where we denote `star(a, b)` as `a star b` using infix notation
commutative binary operation
A binary operation on a set `S` such that for any `a, b in S`, then `a star b = b star a` is called commutative
associative binary operation
A binary operation on a set `S` such that for any `a, b, c in S`, then `(a star b) star c = a star (b star c)` is called associative
bracket placement doesn't matter for associative binary operations
Suppose that `star` is an associative binary operation, then for any `n in ZZ^(ge 3)`, then no matter how one places the brackets in the product `x_1 star x_2 star ... star x_n` it always evaluates to the same value

composition law
A composition law is an associative binary operation
group
a group is a set `G` with a composition law `star` and a distinguished element `e` satisfying the following axioms:
  • identity: `a star e = e star a = a` for all `a in G`
  • inversion: for every `a in G` we have some `b in G` such that `a star b = b star a = e`

note: informally this says that you can do nothing, and you can also undo anything

identity element is unique in a group
Suppose `G` is a group, then the identity element `e` is unique

inverse
Given a group `G` and `a in G`, we denote the unique element `b` such that `a star b = b star a = e` as `a^(-1)` and call it the inverse of `a`
inverse of ab
Given a group `G` and `a, b in G`, then `(a star b)^(-1) = b^(-1) star a^(-1)`

order of a group
The order of a group `G` denoted by `abs(G)` is the cardinality of `G`
abelian
Given a group, if it's composition law is commutative, then we say that the group is abelian
multiplicative notation

Given a group `(G, star)` then since there is only one operation in question, instead of writing the operator between every two pairs of elements we may limit it to `a b`, to refer to `a star b` to reduce visual clutter

note: since `*` is also a simple thing to write, we may also use it in place of `star`

power of an element
Given a group `G` and `a in G`, then we define `a^n := obrace(a a ... a)^("n times")` for any `n in ZZ^(gt 0)` and `a^0 := e`, when `n in ZZ^(lt 0)` then `a^n := (a^(-1))^n`, thus we've defined `a^n` for all `n in ZZ`
power properties
For all `a, b in G`, `a^n a^m = a ^(n + m)` and `(a^n)^m = a^(nm)` for any `m, n in ZZ`
