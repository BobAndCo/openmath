binary operations and groups

binary operation
Let `S` be a set, a binary operation on `S` is a function `star : S xx S to S` where we denote `star(a, b)` as `a star b` using infix notation
commutative binary operation
A binary operation on a set `S` such that for any `a, b in S`, then `a star b = b star a` is called commutative
associative binary operation
A binary operation on a set `S` such that for any `a, b, c in S`, then `(a star b) star c = a star (b star c)` is called associative
bracket placement doesn't matter for associative binary operations
Suppose that `star` is an associative binary operation, then for any `n in ZZ^(ge 3)`, then no matter how one places the brackets in the product `x_1 star x_2 star ... star x_n` it always evaluates to the same value

composition law
A composition law is an associative binary operation
group
a group is a set `G` with a composition law `star` and a distinguished element `e` satisfying the following axioms:
  • identity: `a star e = e star a = a` for all `a in G`
  • inversion: for every `a in G` we have some `b in G` such that `a star b = b star a = e`

note: informally this says that you can do nothing, and you can also undo anything
