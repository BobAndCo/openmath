set
A set is a collection of distinct objects which we call elements of the set. A set can have a finite number of elements or infinitely many elements.
declaration of a finite set
We may declare a finite set by listing it's elements between curly braces like so `{a, b, c, d}` or `{1, 2, 3, 41}`

Suppose we wrote out the set `{a, b, c, d}` and then later on realized that `a = b = c = d`, then this set is simply `{a, a, a, a}`, which we define to just be `{a}`. Therefore we allow duplicates when declaring a set, but realize the set it generates has no such duplicate, so `{1, 1, 1} = {1, 1} = {1}`
set builder notation
Suppose that `P(x)` represents a statement which depends on the variable `x`, then we define `{x in S: P(x)}` to be the set of elements of `S` such that the statement `P` holds true.
empty set
The set with no elements is called the empty set and is denoted by `O/`
element of
Suppose that `a` is an element of a set `S`, then we write `a in S`
set equality
Given two sets `A, B`, we say that `A` and `B` are equal and write `A = B` when `x in A` if and only if `x in B`
subset
Given the sets `A, B`, we say that `A` is a subset of `B` when for every `a in A`, `a` is also in `B`. When this is the case we write `A sube B`
superset
Given the sets `A, B`, we say that `A` is a superset of `B` when for every `b in B`, `b` is also in `A`. When this is the case we write `A supe B`
set equality by subsets
Suppose that `A, B` are sets such that `A sube B` and `B sube A`, then `A = B`
show

We will show the sets are equal using the definition of set equality.

To show that a bi-implication is true, we have to prove both directions. So first suppose that `x in A`, then since `A sube B`, `x in B`. For the other direction suppose that `x in B`, then since `B sube A` then `x in A`. Therefore we know that `x in A` if and only if `x in B` as needed.

union
Given two sets `A, B sube X`, then the union of `A` and `B` is defined as the set `A uu B := {p in X: p in A text( or ) p in B }`
intersection
Given two sets `A, B sube X`, then the intersection of `A` and `B` is defined as the set `A nn B := {p in X: p in A text( and ) p in B }`
disjoint sets
Given two sets `A,B` we say that `A` and `B` are disjoint when `A nn B` = `O/`
pairwise disjoint sets
Let `n in NN_1` and suppose that `S_1, S_2, S_3, ..., S_n` are sets, then these sets are pairwise disjoint if given `i, j in {1, 2, ..., n}` if `i ne j` then `S_i nn S_j = O/`

union of subsets is still a subset
Suppose that `cc C` is a collection of subsets of `X`, then `bigcup_{C in cc C} C sube X`
details Let `x in bigcup_{C in cc C} C`, then `x in U` for some `U in cc C`, since `U` must be a subset of `X` then x is also in `X`. As needed
a set covered in subsets is a union
Suppose `A` is a set and that for each `a in A`, there is a `B_a` such that `a in B_a sube A`, then `A = bigcup_{a in A} B_a`
show

We can see that `cc C = {B_a: a in A}` is a collection of subsets of `A`, so since a union of a subsets is still a subset we have `bigcup_{a in A} B_a sube A`.

But also given `p in A` we know that `p in B_p` so `p in bigcup_{a in A} B_a` which shows `A sube bigcup_{a in A} B_a`, so we can conclude `A = bigcup_{a in A} B_a`.

cartesian product
Suppose that `A, B` are sets, then we define `A xx B := {(a, b): a in A, b in B}`
intersection and cartesian product commute
Given sets `A,B,C,D`, then `(A xx B) nn (C xx D) = (A nn C) xx (B nn D)`
show

We will prove this true by the definition of set equality.

Suppose that `(x,y) in (A xx B) nn (C xx D)`, which is true iff `(x,y) in (A xx B)`, so that `x in A` and `y in B`.

We also know that `(x,y) in (C xx D)` which is equivalent to `x in C` and `y in D`

We have `x in A nn C` and `y in B nn D` which is equivalent to `(x,y) in (A nn C) xx (B nn D)`.

Thus we've proven that `(x,y) in (A xx B) nn (C xx D)` if and only if `(x, y) in (A xx B) nn (C xx D)` as needed.

cartesian product distributes over intersection
Given sets `A, B, C`, then `(A nn B) xx C = (A xx C) nn (B xx C)`
show We will show their equality directly

So suppose that `(x, y) in (A nn B) xx C`, which is true iff `x in A nn B` and `y in C`, which is true iff `(x, y) in A xx C` and `(x, y) in B xx C`, which is true iff `x in (A xx C) nn (B xx C)`.

Therefore `(x, y) in (A nn B) xx C` if and only if `x in (A xx C) nn (B xx C)`, so `(A nn B) xx C = (A xx C ) nn (B xx C)` as needed.
