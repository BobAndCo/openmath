intersected topology
Let `X` be a topological space with toplogy `cc T`. If `Y` is a subset of `X`, then we define the the toplogy `cc T` intersected with `Y` as the set
`cc T_(nn Y) := {Y nn U : U in cc T}`
intersected topology is a topology
The set `cc T_(nn Y)` is a topology on `Y`

We'll show that `O/` and `Y` are elements of `cc T_(nn Y)`, this can be seen as since `O/ in cc T`, then `Y nn O/ = O/ in cc T_(nn Y)` and also since `Y sube X`, then `Y nn X = Y` so `Y in cc T _ (nn Y)`

Now we want to prove that `cc T _(nn Y)` is closed under abitrary unions, which we know because `bigcup_(alpha in I) (U_alpha nn Y) = (bigcup_(alpha in I) U_alpha) nn Y`. Similarly we can see that it's closed under finite intersections as `(U_1 nn Y) nn ... nn (U_n nn Y) = (U_1 nn ... nn U_n) nn Y`.

Therefore we know that `cc T_(nn Y)` is a topology.

subspace topology
