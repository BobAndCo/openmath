sample space
A sample space `Omega` is a non-empty set
probability measure
Given a sample space `Omega` and a function `P: Omega to [0, 1]` , then we say that `P` is a probability if the following hold
  • `P(Omega) = 1`
  • `AA E_1, E_2, ... sube Omega` such that `E_i nn E_j = O/` `AA i ne j in NN_1`, then
    `P(bigcup_(i = 1)^oo E_i) = sum_(i = 1)^oo P(E_i) `
