upper bound
Given `S sube P` where `P` is a partial order, then `u in P` is said to be an upper bound iff `AA s in S, s le u`
lower bound
Given `S sube P` where `P` is a partial order, then `l in P` is said to be an lower bound iff `AA s in S, l le s`
infimum
The infimum of a subset `S` of a partially ordered set `P` is the greatest element in `P` that is a lower bound and is denoted by `text(inf)(S)`. Symbolically this means that `AA s in S, s ge text(inf)(S)`,
infimum or supremum of a set may not be part of itself
Given a set `S` it's possible that `text(inf)(S) notin S` or `text(inf)(S) notin S`
Consider the set `S = (0, 1) sube RR`, then `text(sup)(S) = 1, text(inf)(S)=0` and `1 notin S, 0 notin S`
