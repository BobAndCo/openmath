openmath

welcome to openmath a fully open source project to create a math learning resource to everyone

principles

there are already many online resources to learn math, each one having at least something that could be considered annoying

  • youtube - not suitable for offline use, contributing requires computer with video editing software
  • wikipedia - contains extranous information, goal is for general information
  • proofwiki - less of a tool to learn topics more of a proof database, mediawiki tech stack
  • nlab - higher level mathematics less accessible for beginners, mediawiki
  • brilliant - nice entry for beginners lacks cumulative content, payment required
  • textbooks - most of them are freely accessible online, but are not easy to navigate (index to page number workflow)
  • libretexts - quite useful, users cannot easily contribute, complex tech stack

what openmath tries to do is as follows

  • focus on learning topics in mathematics
  • suitable for offline use
  • contains concise mathematical information
  • simple tech stack, no database
  • accessible for all skill levels
  • easy to navigate
  • freely accessible
  • any user can contribute
  • write explanatory proofs
    • when reading a proof you may be able to understand that it's true by following along, then a second analysis is required to figure out how they have come up with the proof, we will try to write explanatory proofs which do both at the same time.

additionally it focuses on recursive learning, which is that given a statement for each definition in that statement you can follow it back to a previous definition recursively until you're at some fundamental definition and then work your way back to the original statemen. openmath attempts to make recursive learning as smooth as possible
