relation
A relation on two sets `X, Y` is a subset of `X xx Y`
function
a relation that has no repeated (TODO: rigor) x values is said to be a function.

Functions are usually denoted by the symbols `f,g,h,...` and if `(x,y) in f` then we say that `f(x) = y` and we can think of `f` mapping `x` to `y`

set filter
Given a set `S` and a unary relation `star: S to S` we can write `RR^{star}` to denote the set `{x in S: star(x)}`.

We can take binary operations and load a value in to produce a unary operation to write something like `RR^{!= 0}` which denotes the set of real numbers that are non-zero.

open set
A set `U` is open in `RR` when for every point `x in U` there is some `epsilon in RR^(>0)` such that `(x - epsilon, x + epsilon) sube U`
neighbourhood of a point
given a point `x in RR` a neighborhood of `x` is a set of the form `(x - epsilon, x + epsilon)` where `epsilon in RR^{gt 0}`, in other words it's an open set centered at `x`
limit of a function at a point
given a function`f: RR to RR`, we say that the limit of `f` as `x to a` equals `L` when the following is satisfied
`AA epsilon in RR^(gt 0), EE delta in RR^(gt 0), text(such that) AA x in text(dom)(f), abs(x - a) lt delta => abs(f(x) - L) lt epsilon`
absolute value
the absolute value function `| * | : RR to RR^{>=0}` is defined by
`abs(x) = {(x text( if ) x > 0), (-x text( if ) x < 0):}`
multiplication in absolute value
`|x * y| = |x| * |y|`
  • If both `x, y` are positive then `abs(x * y) = x * y = abs(x) * abs(y)`
  • if both `x, y` are negative, then their product is positive so it follows as above
  • if one of `x,y` is positive and the other negative, without loss of generality assume that `x < 0` and `y > 0`then their product is negative so `abs(x * y) = -1 * x * y = abs(x) * abs(y)`
  • if any one of `x, y` is zero, then `abs(x * y) = 0 = abs(x) * abs(y)`
multiplying by minus one does nothing in absolute value
`AA x in RR, |x| = |-x|`
absolute value is greater than itself
`forall x in RR, x <= |x|`
  • if `x > 0` then `abs(x) = x`
  • if `x < 0` then `abs(x) = -x` so `abs(x) ge -x > x` this is true since `x < 0` and `0 < -x`
absolute value is greater than the negative version of itself
`forall x in RR, -x <= |x|`
using the fact `y < abs(y)` and subbing in `-x` we obtain the result
`|a|^-1 = |1/a|`
`|a| * |1/a| = |a * 1/a| = |1| = 1`

power series
a power series is an infinite series of the form
`sum_{n = 0}^oo a_n(x - c)^n`
where `a_n, c in RR`
taylor series
the taylor series for a function `f : A sube RR to RR` that is `C^oo` at `a in RR` is given by
`sum_{n = 0}^oo (f^{(n)}(a)) / (n!) (x - a)^n`
where `f^{(n)}(a)` is the `n`-th derivative of `f` evaulated at the point `a`
taylor's
Let `k >= 1` be an integer and let the function `f: RR -> RR` be k times differentiable at the point `a in RR`. Then there exists a function `h_k: RR -> RR` such that

`f(x) = f(a) + f^'(a) * (x - a) + f^('')(a)/(2!) (x - a)^2 + ... + f^{(k)}(a)/(k!)(x - a)^k + h_k(x) (x - a)^k`

where `lim_{x -> a} h_k(x) = 0`
real analytic
we say that a function `f` is real analytic on an open set `D sube RR` if given any point `x_0 in D` we can write it as a taylor series
`f(x) = sum_{n = 0}^oo (f^{(n)}(x_0)) / (n!) (x - x_0)^n`
where the above equality holds for any `x` in a neighborhood of `x_0`
taylor's expansion
Let `I sube RR` be an open interval, suppose `f: I to RR` is real analytic on `I`,so that for every `a in I` then there exists some `r in RR^{>0}` with `(a - r, a + r) sube RR` and for every `x in (a-r, a+r)` we have
`f(x) = sum_{n = 0}^oo (f^{(n)}(a)) / (n!) (x - a)^n`
And we denote `r` as the radius of convergence.

Since we know that `f: I to RR` is real analytic, we know that for every point `a in I`, there is some neighborhood `U` of `a` where the equality `f(x) = sum_{n = 0}^oo (f^{(n)}(x_0)) / (n!) (x - x_0)^n` holds

By the definition of neighbourhood, we know that `U = (a - epsilon, a + epsilon)` for some `epsilon in RR^{gt 0}`.

Therefore we can take `r = epsilon`

closed form of a geometric series
`sum_{n = 0}^k x^n = (1 - x^(k+1)) / (1 - x)`
Let `S = sum_{n = 0}^k x^n`, then `xS - S = x^(k+1) - x^0` therefore `S = (x^(k+1) - 1)/(x - 1)`, so also `S =(1 - x^(k+1))/(1 - x)` by factoring -1 from the numerator and the denominator, as needed
exponent limit
if `abs(r) < 1`, then `lim_{k to oo} r^k = 0` otherwise it diverges to infinity
geometric power series
for `abs(x) < 1` we have `sum_{n = 0}^oo x^n = 1/(1 - x)`
details By definition we know that `sum_{n = 0}^oo x^n = lim_{k to oo} sum_{n = 0}^k x^n` and by the closed form of the geometric series we know that `lim_{k to oo} sum_{n = 0}^k x^n = lim_{k to oo}(1 - x^(k+1))/(1-x)` thus by the exponent limit lemma if `abs(x) < 1` then `lim_{k to oo} x^(k+1)` becomes zero and we get `lim_{k to oo} (1 - x^(k+1))/(1-x) = 1/(1-x)`
symmetric integral of an even function
Suppose `a in RR` and `f` an even function then
`int_-a^a f(x) dx = 2 * int_0^a f(x) dx`
improper integral
If `f` is continuous on `[a, oo)`, then the improper integral of `f` over `[a, oo)` is denoted and defined by
`int_a^oo f(x) dx := lim_{R to oo} int_a^R f(x) dx`
double improper integral
If the integrals `int_-oo^a f(x) dx` and `int_a^oo f(x) dx` are convergent, then the improper integral of `f` over `(-oo, oo)` is defined and notated as follows
`int_-oo^oo f(x) dx := int_-oo^a f(x) dx + int_a^oo f(x) dx`
symmetric evaluation of a doubly improper integral

If `int_-oo^oo f(x) dx` is convergent, then it can be evaluated using the following limit

`lim_{R to oo} int_-R^R f(x) dx`

If `int_-oo^oo f(x) dx` is convergent, then so are `int_-oo^a f(x) dx` and `int_a^oo f(x) dx`.

These being convergent means that `lim_{R -> oo} int_a^R f(x) dx` and `lim_{K to -oo} int_k^a f(x) dx` exist. Since we can replace dummy variables (TODO) with anything we'd like, we can also replace `K` with `R` so that `lim_{R to -oo} int_R^oo f(x) dx` also exists.

By the limit sum rule, we can see that

`int_-oo^oo f(x) dx := lim_{K to -oo} int_K^a f(x) dx + lim_{R to oo} int_a^R f(x) dx ` `= lim_{R to oo} int_-R^a f(x) dx + lim_{R to oo} int_a^R f(x) dx ` `= lim_{R to oo} (int_-R^a f(x) dx + int_a^R f(x) dx)` `= lim_{R to oo} int_-R^R f(x) dx`

indefinite integral comparison test
if `f(x) >= g(x) >= 0` on the interval `[a, oo)` then
  1. if `int_a^oo f(x) dx` converges then so does `int_a^oo g(x) dx`
  2. if `int_a^oo g(x) dx` diverges then so does `int_a^oo f(x) dx`
open interval
any set set `I sube RR` of the form
`{x in RR : a < x < b}`
for `a, b in RR` with `a < b`. It may be notated by `(a, b)`
closed interval
any set set `I sube RR` of the form
`{x in RR : a <= x <= b}`
for `a, b in RR` with `a <= b`. It may be notated by `[a, b]`
alternate words for set
family, collection both refer to sets, the exist so that when we are dealing with sets of sets, we can write a collection or family of sets instead to be less confusing
half-open interval
any set set `I sube RR` of the form
`{x in RR : a <= x < b}` or `{x in RR : a < x <= b}`
for `a, b in RR` with `a <= b`. It may be notated by `[a, b)` or `(a, b]` respectively
interval
an interval is any set which is an open, closed, or half-open interval
length of an interval
given an interval `I` with endpoints `a, b` the length is defined as `b - a` and is denoted by `text(len)(I)`
countable
a set `S` is countable if it is finite or there is a bijection `C: S -> NN`. If there is then we can say that `S` is enumerable, as we can assign counting numbers to each element of the set.
cover
a cover for a set `X` is a family `cc F` of sets such that `X sube uuu_{F in cc F} F `, we may also say that `cc F` is a covering of `X`
measure 0
A subset `N` of `RR` is said to have measure `0` if for each `epsilon > 0` there is a countable family `cc H = {I_1, I_2, ...}` of open intervals covering `N` with `sum_{k in NN} text(len)(I_k) < epsilon`
Lebesgue's Criterion for integrability
let `f: [a, b] to RR`, then `f` is Riemann integrable if and only if `f` is bounded and the set of discontinuities of `f` has measure 0.
