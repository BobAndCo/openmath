complex

addition
Given `a + bi, c + di in CC`, we define `(a + bi) + (c + di) = (a + c) + (b + d) i`
complex multiplication
given `a + bi, c + di in CC`, then:
`(a + bi) * (c + di) = ab + i (ad + bc) - db = (ad - bd) + i (ad + bc)`
equality
Given `a + bi, c + di in CC`, they are equal when `a = c` and `b = d`
conjugate
suppose `z = x + iy`, then the conjugate is denoted by
`bar z := x - iy`
conjugate cancels
`bar bar z = z`
show Let `z = x + iy`, then `bar z = x - iy = x + i (-y)` then `bar bar z = x - i (-y) = x + iy = z` as needed.
modulus
for any `z = x + iy in CC`, we define
`abs(z) := sqrt(x^2 + y^2)`
modulus ignores conjugates
`abs(bar(z)) = abs(z)`
show Suppose that `z = x + iy`, then `abs(bar(z)) = abs(x - iy) = sqrt(x^2 + (-y)^2) = sqrt(x^2 + y^2) = abs(z)`
complex number times it's conjugate equals the modulus squared
`z * bar z = |z|^2`
show suppose that `z = x + iy`, then `z * bar z = (x + iy) * (x - iy) = x^2 + y^2 + xyi - xyi = x^2 + y^2`, but then again `abs(z) = abs(x + iy) = sqrt(x^2 + y^2)` thus `z * bar z = abs(z)^2`
inverse
the inverse of a complex number `z in CC` is another `w in CC` such that `z * w = 1`, we denote `w` by `z^(-1)` or `1/z`
value of the inverse
Given `z in CC` we have `z^(-1) = bar(z) / |z|^2`
show We know that `z * bar z = |z|^2`, therefore `z * bar(z) / |z|^2 = 1`, so by the definition of inverse `z^(-1) = bar(z) / |z|^2` as needed.
For any `z in CC, | z / bar(z) | = 1`
show

By the value of the inverse we have: `1/bar(z) = bar(bar(z)) / |bar(z)|^2` since we know that two complex conjugates cancel and modulus ignores conjugates, then `bar(bar(z)) / |bar(z)|^2 = z / |z|^2`

Now `abs(z / bar z) = abs(z (z / abs(z)^2)) = abs((z/abs(z))^2) = abs(z/abs(z)) * abs(z/abs(z)) = 1 * 1 = 1` as needed

real part of a complex number
given `z = x + iy in CC` we say that `x` is the real part of `z` and define the function
`cc R(z) = x`
imaginary part of a complex number
given `z = x + iy in CC` we say that `y` is the imaginary part of `z` and define the function
`cc I(z) = y`
extracting the real part of a complex number
suppose `z in CC`, then `fr R(z) = (z + bar(z))/2`
modulus is greater than it's components
for any `z in CC` we have both
  • `abs(fr R (z)) le abs(z)`
  • `abs(fr I (z)) le abs(z)`
imaginary part distributes
`fr I(z + w) = fr I(z) + fr I(w)`
modulus plus one of it's components is positive
Suppose that `f in {cc R, cc I}`, then
`abs(z) + f(z) ge 0`
Let `z = x + iy in CC` then suppose without loss of generality that `f = cc R`, then
`-x le abs(x) = sqrt(x^2) le sqrt(x^2 + y^2) = abs(x + iy)`
therefore `0 le x + abs(x+ iy)` which means `0 le cc R(z) + abs(z)`
creating squares
without use of the triangle inequality prove that `abs(z - 1)^2 = abs(z)^2 + 1 - 2 cc R(z)`, then conclude that `abs(z - 1) le abs(z) + 1`

`abs(z - 1)^2` = `(z - 1) * bar(z - 1) = (z - 1) * (bar z - 1) = z * bar z - z - bar(z) + 1`, now recall that `cc R(z) = (z + bar(z)) / 2` so we can continue the equality with `abs(z)^2 - 2 cc R(z) + 1`, showing the first part

for the second part, we can notice that `(abs(z) + 1)^2 = abs(z)^2 + 2abs(z) + 1`, so then `abs(z - 1)^2 = abs(z)^2 + 1 - 2 cc R(z) = abs(z)^2 + 2 abs(z) - 2 abs(z) + 1 - 2 cc R(z) = (abs(z) + 1)^2 - 2 ( abs(z) + cc R(z))`, then by this proposition we can see that the last term in the previous equality chain is less than or equal to `(abs(z) + 1)^2` allowing us to conclude that `abs(z - 1)^2 le (abs(z) + 1)^2` which implies that `abs(z - 1) le abs(z) + 1`
