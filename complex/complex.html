analytic function
a function `f` is analytic if it can be represented by a power series
power series
a power series around `z_0 in CC` is a function of the following form `f(z) = sum_{k = 0}^oo a_k * (z - z_0)^k`
convergence of a power series
Suppose that `f(z) = sum_{k = 0}^oo a_k * (z - z_0)^k` is a power series, then there exists a unique `r in RR uu {oo}` known as the radius of convergence such that:
  • if `|z - z_0| < r` then the given power series converges
  • if `|z - z_0| > r` then the power series diverges
a power series is an analytic function on the inside of it's circle of convergence
coefficients of a power series
Let `f(z) = sum_{n = 0}^oo a_n * (z - z_0)^n` be the analytic function defined on the inside of the circle of convergence of the given power series. Then `f'(x) = sum_{n = 1}^{oo} n * a_n (z - z_0)^(n - 1)` and this series has the same circle of convergence as `sum a_n (z - z_0)^n`, moreover the cofficents `a_n` are given by `a_n = f^{(n)}(z_0) /(n!)`
power series expansions around the same center are unique
taylors theorem
let `f` be analytic on an open set `A sube CC`, let `z_0 in A` and let `A_r = {z in CC : |z - z_0| < r} sube A`, then for every `z in A_r` the series

`sum_{n = 0}^oo f^{(n)}(z_0)/(n!) * (z - z_0)^n`

converges on `A_r` (`r` is it's radious of convergence) And

`f(z) = sum_{n = 0}^oo f^{(n)}(z_0)/(n!) * (z - z_0)^n`

analytic iff power series
Let `A` be a region in `CC` and let `f` be a complex valued function defined on `A`. Then `f` is analytic on `A` iff for each `z_0` in `A` there is a number `r > 0` such that the disk `D(z_0, r) sube A` and `f` equals a convergent power series on `D(z_0, r)`
zeros of analytic functions
suppose `f: Omega -> CC` is analytic on an open set `omega in CC` and that `c in Omega`. We know taht the taylor series for `f(z)` centered at `c` converges to `f(z)` at least in the largest open disk `D(c, r)` centered at `c` and contained in `Omega`. Therefore we have:

`f(z) = sum_{k = 0}^{oo} f^{(k)}(c) / (k!) (z - c)^k`

for `|z - c| < r`.
  • If `f^{(k)}(c) = 0` for all `k`, then `f(z)` is identically `0` in `D(c, r)`
  • If not, then there is a smallest non-negative integer `n` holding `f^{(n)}(c) != 0`. If `n = 0`, then `f(c) !=0`. If `n > 0`, then `f(c) = f'(c) = f''(c) = ... = f^{(n - 1)}(c) = 0` but `f^{(n)}(c) != 0` and we say that `f` has a zero of order `n` at `c`. From algebra we know that a polynomial has a zero at `c` iff `z - c` is a factor of that polynomial. The zero has order of multiplicity `n` if the factor `z - c` occurs exactly `n` times. Consideration of the Taylor series shows that an analytic function has a zero at `c` iff `z - c` is a factor of that function and the multiplicity is defined in the same way. For `z` in `D(c, r)`, then since the first `n - 1` terms wer zeros then the power series becomes

    `f(z) = a_n* (z - c)^n + a_(n + 1) (z - c)^(n + 1) + + a_(n + 2) (z - c)^(n + 2) + ... = (z - c)^n * [a_n + a_(n + 1) * (z - c) + a_(n + 2) * (z - c)^2 + ...]`

    where `a_k = f^{(k)}(c)/(k!)`. The power series in the square brackets converges in `D(c, r)` to an analytic function `phi(z)` TODO: why? with `phi(c) = a_n = f^{(n)}(c)/(n!) != 0`. Thus `f` has a zeor of order `n` at `c` iff `f(z)` can be factored in a neightborhood of `c` as `f(z) = (z - c)^n phi(z)` where `phi` is analytic in a neighborhood of `c` and `phi(c) != 0`.
isolated zeros
looking at the factorization in the last theorem, we can see that `phi(z)` is analytic and so is continuous on `D(c, r)`, since `phi(c) != 0`, there is a radius `p` with `0 < p < r` such that `phi(z)` is never 0 for `|z - c| < p`. Since `f(z) = (z - c)^n phi(z)` for all `z in D(c, p)` we conclude that `f` has no zeros other than `c` in that disk, in this sense the zeros of `f` are isolated.
laurent expansion theorem
Let `0 <= r_1 < r_2` and `z_0 in CC`, and consider the region `A = {z in CC | r_1 < |z - z_0| < r_2}`. We allow `r_1 = 0` or `r_2 = oo` or both. Let `f` be analytic on the region `A`. Then we can write

`f(z) = sum_{n = 0}^oo a_n(z - z_0)^n + sum_{n = 1}^oo b_n/(z-z_0)^n`

where both series on the right side of the equation converge absoltely on `A` and uniformly in the sets of the form `B_{p_1, p_2} = {z in CC : p_1 <= |z - z_0| <= p_2} ` where `r_1 < p_1 < p_2 < r_2`
